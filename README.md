# Paradise Nursery 🌿

Paradise Nursery es una aplicación web moderna de comercio electrónico diseñada para amantes de las plantas. Este proyecto forma parte de mi certificación en **React** y demuestra habilidades en la gestión de estado global, diseño responsivo y componentes reutilizables.

## 🚀 Características
- **Página de Inicio (Landing Page):** Una bienvenida atractiva con acceso directo al catálogo.
- **Catálogo de Productos:** Visualización de plantas con filtros por categorías (Aire Puro, Aromáticas, etc.).
- **Carrito de Compras:** Gestión en tiempo real (añadir, eliminar, aumentar/disminuir cantidades).
- **Cálculo Automático:** Total de artículos y costo total actualizados dinámicamente.
- **Diseño Glassmorphism:** Interfaz moderna con efectos de transparencia y desenfoque.

## 🛠️ Tecnologías
- **React.js** (Vite)
- **Redux Toolkit** (Gestión de estado global)
- **React Router** (Navegación entre páginas)
- **CSS3** (Variables y diseño responsivo)
- **Lucide React** (Iconografía)
